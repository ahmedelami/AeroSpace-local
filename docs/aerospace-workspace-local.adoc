== Synopsis
[verse]
// tag::synopsis[]
aerospace workspace-local [-h|--help] <slot-index>

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
`workspace-local` focuses the workspace that occupies the given slot on the monitor
that currently owns the keyboard focus.

* Slot indexes are 1-based and local to each monitor.
* If the requested slot is empty, a stub workspace is created for that monitor so
  the slot can be focused.
* When `[general] workspace_mode` is not `per-monitor`, the command fails with a
  descriptive error rather than silently falling back to global semantics.
