// =========================================================== Description
== Description

// tag::body[]
Swaps the currently visible workspaces between two monitors. The monitors can be
specified explicitly (using the IDs from `list-monitors`) or relatively (`focused`,
`mouse`, `next`, `prev`, `left`, `right`, `up`, `down`). Slot assignments inside
`WorkspaceLocalIndexing` are updated so the swap persists.

When `[general] workspace_mode = "per-monitor"`, the swap keeps each monitorâ€™s local
slots intact; in global mode it behaves like a convenience wrapper around
`move-workspace-to-monitor`.
